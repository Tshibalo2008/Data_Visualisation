# Data_Visualisation
Bright Shopping Trends Analysis

# Introduction
Bright Shopping Trends Analysis is a data visualization and analytics project focused on understanding customer purchasing behavior.

Using Google Looker Studio (formerly Data Studio),

This project explores how different demographic factors such as age group, gender, and payment methods influence shopping patterns.

#
The dashboard provides key insights into:

Payment methods by age group and preferred transaction types

Customer demographics and spending habits

Trends over time in purchase frequency and value

Visual comparisons using interactive charts (donut, stacked bar, and trend visuals)

# Goal
The goal of this project is to transform raw transactional data into actionable business intelligence, 
helping retailers understand what drives customer choices and optimize marketing strategies.

# Tools & Technologies:
Google Looker Studio (Dashboard & Data Visualisation)
Excel (Data Preparation)
